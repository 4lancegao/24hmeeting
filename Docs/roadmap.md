# 24hMeeting Roadmap

## Phase 1: MVP（当前仓库）

- [x] iOS + macOS 共用 UI
- [x] 24 小时会议上限控制
- [x] 会话状态管理
- [x] 视频引擎抽象接口

## Phase 2: RTC 接入

- [ ] 接入真实视频 SDK（Agora / WebRTC）
- [ ] 设备权限（麦克风 / 摄像头）
- [ ] 本地音视频预览
- [ ] 网络切换重连

## Phase 3: 服务端

- [ ] 房间服务（创建、销毁、鉴权）
- [ ] 信令服务（WebSocket）
- [ ] 参会者管理（上麦、静音、踢人）
- [ ] 会议事件日志

## Phase 4: 生产质量

- [ ] 崩溃与性能监控
- [ ] 音视频质量指标（MOS、RTT、丢包）
- [ ] 自动化测试（UI + 集成）
- [ ] 灰度发布
